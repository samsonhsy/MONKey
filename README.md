# MONKey

[1]: <https://github.com/samsonhsy/MONKey/>

MONKey is an Android-based application using scheduled app-blocking to help users retain focus.

### Getting Started

Install <a href="https://developer.android.com/">Android Studio</a>, at least version and build locally.



### Future Updates

* Add reward system to encourage users using MONKey actively.
* Add NFC detection for unlock tasks.

### Dependencies

- Android Studio Otter 3 Feature Drop | 2025.2.3
- minSdk 26
- targetSdk 36
- compileSdk 36
- Java 11
