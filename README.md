# MONKey

[1]: <https://github.com/samsonhsy/MONKey/>

MONKey is an Android-based application using scheduled app-blocking to help users retain focus.

### Built With

- <a href="https://developer.android.com/compose">Jetpack Compose</a>

### Getting Started

Install <a href="https://developer.android.com/">Android Studio</a> with the latest version and build locally.

After a successful build, go to the MONKey settings page and enable services.

#### Reminder
It is recommended to set MONKey battery usage to unrestricted and allow Auto-start if available.
